7''' Common enumerations to be used together with |Enum| property. 

8 

9This module provides many pre-defined enumerations, as well as functions 

10for creating new enumerations. 

11 

12New enumerations can be created using the |enumeration| function: 

13 

14.. code-block:: python 

15 

16 #: Specify a nautically named side, port or starboard 

17 MyEnum = enumeration("port", "starboard") 

18 

19Typically, enumerations are used to define |Enum| properties: 

20 

21.. code-block:: python 

22 

23 from bokeh.model import Model 

24 from bokeh.core.properties import Enum 

25 

26 class MyModel(Model): 

27 

28 location = Enum(MyEnum, help=""" 

29 Whether the thing should be a port or starboard. 

30 """) 

31 

32Enumerations have a defined order and support iteration: 

33 

34.. code-block:: python 

35 

36 >>> for loc in MyEnum: 

37 ... print(loc) 

38 ... 

39 port 

40 starboard 

41 

42as well as containment tests: 

43 

44.. code-block:: python 

45 

46 >>> "port" in MyEnum 

47 True 

48 

49Enumerations can be easily documented in Sphinx documentation with the 

50:ref:`bokeh.sphinxext.bokeh_enum` Sphinx extension. 

51 

52---- 

53 

54.. autofunction:: bokeh.core.enums.enumeration 

55 

56---- 

57 

58.. |Enum| replace:: :class:`~bokeh.core.properties.Enum` 

59.. |enumeration| replace:: :func:`~bokeh.core.enums.enumeration` 

60 

61'''

151class Enumeration(object): 

152 ''' Represent an enumerated collection of values. 

153 

154 .. note:: 

155 Instances of ``Enumeration`` typically should not be constructed 

156 directly. Instead, use the |enumeration| function. 

157 

158 ''' 

159 __slots__ = () 

160 

161 def __iter__(self): 

162 return iter(self._values) 

163 

164 def __contains__(self, value): 

165 if not self._case_sensitive: 

166 value = value.lower() 

167 return value in self._values 

168 

169 def __str__(self): 

170 if self._quote: 

171 return "Enumeration(%s)" % ", ".join(repr(x) for x in self._values) 

172 else: 

173 return "Enumeration(%s)" % ", ".join(self._values) 

174 

175 def __len__(self): 

176 return len(self._values) 

177 

178 __repr__ = __str__ 

179 

180def enumeration(*values, **kwargs): 

181 ''' Create an |Enumeration| object from a sequence of values. 

182 

183 Call ``enumeration`` with a sequence of (unique) strings to create an 

184 Enumeration object: 

185 

186 .. code-block:: python 

187 

188 #: Specify the horizontal alignment for rendering text 

189 TextAlign = enumeration("left", "right", "center") 

190 

191 Args: 

192 values (str) : string enumeration values, passed as positional arguments 

193 

194 The order of arguments is the order of the enumeration, and the 

195 first element will be considered the default value when used 

196 to create |Enum| properties. 

197 

198 Keyword Args: 

199 case_sensitive (bool, optional) : 

200 Whether validation should consider case or not (default: True) 

201 

202 quote (bool, optional): 

203 Whether values should be quoted in the string representations 

204 (default: False) 

205 

206 Raises: 

207 ValueError if values empty, if any value is not a string or not unique 

208 

209 Returns: 

210 Enumeration 

211 

212 ''' 

213 if not (values and all(isinstance(value, string_types) and value for value in values)): 

214 raise ValueError("expected a non-empty sequence of strings, got %s" % values) 

215 

216 if len(values) != len(set(values)): 

217 raise ValueError("enumeration items must be unique, got %s" % values) 

218 

219 attrs = {value: value for value in values} 

220 attrs.update({ 

221 "_values": list(values), 

222 "_default": values[0], 

223 "_case_sensitive": kwargs.get("case_sensitive", True), 

224 "_quote": kwargs.get("quote", False), 

225 }) 

226 

227 return type(str("Enumeration"), (Enumeration,), attrs)()
