127class GMapPlot(MapPlot): 

128 ''' A Bokeh Plot with a `Google Map`_ displayed underneath. 

129 

130 Data placed on this plot should be specified in decimal lat/lon coordinates 

131 e.g. ``(37.123, -122.404)``. It will be automatically converted into the 

132 web mercator projection to display properly over google maps tiles. 

133 

134 Note that Google Maps exert explicit control over aspect ratios at all 

135 times, which imposes some limitations on ``GMapPlot``: 

136 

137 * Only ``Range1d`` ranges are supported. Attempting to use other range 

138 types will result in an error. 

139 

140 * Usage of ``BoxZoomTool`` is incompatible with ``GMapPlot``. Adding a 

141 ``BoxZoomTool`` will have no effect. 

142 

143 .. _Google Map: https://www.google.com/maps/ 

144 

145 '''
