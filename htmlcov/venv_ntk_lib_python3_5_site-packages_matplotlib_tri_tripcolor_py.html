11 Create a pseudocolor plot of an unstructured triangular grid. 

12 

13 The triangulation can be specified in one of two ways; either:: 

14

25 tripcolor(x, y, mask=mask, ...) 

26 tripcolor(x, y, triangles, mask=mask, ...) 

27 

28 in which case a Triangulation object will be created. See 

29 :class:`~matplotlib.tri.Triangulation` for a explanation of these 

30 possibilities. 

31 

32 The next argument must be *C*, the array of color values, either 

33 one per point in the triangulation if color values are defined at 

34 points, or one per triangle in the triangulation if color values 

35 are defined at triangles. If there are the same number of points 

36 and triangles in the triangulation it is assumed that color 

37 values are defined at points; to force the use of color values at 

38 triangles use the kwarg ``facecolors=C`` instead of just ``C``. 

39 

40 *shading* may be 'flat' (the default) or 'gouraud'. If *shading* 

41 is 'flat' and C values are defined at points, the color values 

42 used for each triangle are from the mean C of the triangle's 

43 three points. If *shading* is 'gouraud' then color values must be 

44 defined at points. 

45 

46 The remaining kwargs are the same as for 

47 :meth:`~matplotlib.axes.Axes.pcolor`.
