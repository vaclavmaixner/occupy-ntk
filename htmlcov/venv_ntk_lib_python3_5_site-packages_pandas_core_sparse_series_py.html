38class SparseSeries(Series): 

39 """Data structure for labeled, sparse floating point data 

40 

41 Parameters 

42 ---------- 

43 data : {array-like, Series, SparseSeries, dict} 

44 .. versionchanged :: 0.23.0

50 Code for missing value. Defaults depends on dtype.
