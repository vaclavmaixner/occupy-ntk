13:ref:`bokeh.core.enums` 

14 Properties on Bokeh models support automatic type validation, including

19:ref:`bokeh.core.properties` 

20 The fundamental building block of Bokeh apps and documents is the Bokeh 

21 models, for instance plots, ranges, axes, etc. Bokeh models are comrpised 

22 of properties, which are named attributes with specified types. Model 

23 properties can automatically validate and serialize themselves. This 

24 section describes all the property types that can be attached to Bokeh 

25 models, which is of interest when creating custom extensions. 

26 

27:ref:`bokeh.core.property_mixins` 

28 Some collections of properties appear together often. Property mixins 

29 are groups of properties, such as a ``fill_color`` and ``fill_alpha``, 

30 that make up a single unit that can be easily applied to Bokeh models 

31 all at once. 

32 

33:ref:`bokeh.core.validation` 

34 When serializing a Document for use by BokehJS, the Bokeh python

38 to find out more specifics about such warnings and errors.
